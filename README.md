# Smart Deployment Guard – AI Driven DevOps System

## Overview
Smart Deployment Guard is an AI-driven DevOps automation system that prevents risky deployments by checking server health before deployment and taking intelligent decisions.

---

## Project Goal
Ensure safe and reliable deployments by validating system health metrics before allowing deployment.

---

## Features

- CPU Usage Monitoring  
- Memory Usage Validation  
- Disk Usage Check  
- Port Availability Check  
- Rule-Based AI Decision Engine  
- Deployment Allow / Block System  
- Self-Healing Mechanism  
- Logging System for Audit & Debugging  

## Advanced DevOps Features

- Docker image build and container execution
- Kubernetes Deployment and NodePort Service
- Kubernetes liveness and readiness probes
- Kubernetes pod health validation using Python
- GitHub Actions CI with Docker validation
- AI rule-based decision engine for deployment safety
